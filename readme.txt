== Description ==

A simple plugin/widget to display your Vimeo videos. 

== Installation ==


1. Upload `vimeowp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your username in the Vimeo+WP settings page under `Settings`
4. Drag a widget to one of your widgetized areas of your theme
